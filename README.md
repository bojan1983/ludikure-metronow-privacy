# Privacy Policy

**Metro Now** respects your privacy.

## What we collect
We do not collect, store, or share any personal data.

## Location
If you enable location, it is used only to find the nearest Metro station.  
Location data is **not stored** and **not transmitted** to any server.

## Data sources
All transit data is fetched through a lightweight proxy from WMATA public feeds.
